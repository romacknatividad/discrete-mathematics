Course Description

Discrete mathematics provides the foundation for precise reasoning in computing. Students will learn how to write proofs, analyze combinatorial structures, work with graphs and relations, and apply discrete methods to computational problems.

The course emphasizes clarity of argument, correct use of definitions, and the ability to translate between mathematical ideas and computer science applications.

Learning Outcomes

  • Construct and evaluate mathematical proofs using standard proof techniques.
  • Model discrete structures using sets, functions, relations, and graphs.
  • Solve counting problems and analyze recurrence relations.
  • Connect discrete mathematics concepts to algorithms and computation.

Course Contents

Major Topics

Select a topic to focus on the main ideas, subtopics, and representative applications.

Logic and Proof

Students begin with formal statements and valid inference. This unit develops the proof techniques required throughout the rest of the course.

Core Ideas

  • Propositions, predicates, and quantifiers
  • Logical equivalence and implication
  • Direct proof, contradiction, contrapositive, and proof by cases

Applications

  • Program specifications
  • Reasoning about correctness
  • Translating informal claims into formal mathematical statements

Sets, Functions, and Relations

This unit introduces the basic language used to model discrete systems and data relationships.

Core Ideas

  • Set operations and power sets
  • Functions, inverse images, injective and surjective mappings
  • Relations, equivalence relations, and partial orders

Applications

  • Database relationships and state modeling
  • Abstraction of computational structures
  • Classification and partitioning problems

Number Theory and Induction

These topics support recursive reasoning, algorithm analysis, and modular computation.

Core Ideas

  • Divisibility, primes, and modular arithmetic
  • The Euclidean algorithm
  • Weak and strong induction, recursive definitions

Applications

  • Cryptography foundations
  • Recursive algorithm correctness
  • Proofs over sequences and data structures

Counting and Recurrences

Students learn systematic methods for enumerating possibilities and describing recursive growth.

Core Ideas

  • Product and sum rules
  • Permutations, combinations, and binomial coefficients
  • Pigeonhole principle, inclusion-exclusion, and linear recurrences

Applications

  • Algorithm counting arguments
  • Discrete probability preparation
  • Runtime growth patterns in recursive procedures

Graphs and Trees

Graph theory provides a natural language for networks, search, connectivity, and hierarchical structures.

Core Ideas

  • Graphs, paths, cycles, and connectivity
  • Trees, spanning trees, and traversals
  • Representations and basic graph properties

Applications

  • Network modeling
  • Search algorithms
  • Data structures and dependency analysis

Boolean Logic and Formal Models

The course concludes by connecting logical systems with digital computation and abstract machine models.

Core Ideas

  • Boolean algebra and logical identities
  • Switching circuits and simplification
  • Finite-state models and formal reasoning

Applications

  • Digital logic design
  • State-based computation
  • Connections to automata and formal languages
