# Discrete Mathematics GitHub Page

This repository contains a static GitHub Pages site for an undergraduate discrete mathematics course for computer science.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and the `/ (root)` folder.
6. Save.

GitHub Pages will serve `index.html` from the repository root.

## Customize

- Update instructor, email, room, office hours, and course number in `index.html`.
- Replace placeholder resource links in `index.html` with real URLs.
- Adjust colors and layout in `styles.css` if you want a different visual direction.
